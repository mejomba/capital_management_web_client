# GLOSSARY — واژه‌نامه‌ی دامنه

این واژه‌نامه مدل ذهنی پروژه را قفل می‌کند. شناسه‌های انگلیسی همان‌هایی‌اند که در کد/DB/API به کار می‌روند.

## موجودیت‌ها

| مفهوم (فارسی) | شناسه (انگلیسی) | تعریف |
|---|---|---|
| کاربر | `user` | صاحب داده؛ هر داده scope به یک کاربر. |
| حساب | `account` | ظرف نگه‌دارنده‌ی دارایی (صرافی، بانک، کارگزاری، فیزیکی، کیف‌پول، ملک). |
| دارایی/نماد | `asset` | چیزی که نگه‌داری می‌شود: BTC، سهم، گرم طلا، USD، یک ملک خاص. |
| کلاس دارایی | `asset_class` | equity, fund, crypto, metal, forex, real_estate, fiat, other. |
| موجودی | `holding` | مقدار یک دارایی در یک حساب. **همیشه مشتق‌شده از تراکنش‌ها**. |
| تراکنش | `transaction` | رویداد مالی اتمی و تغییرناپذیر. |
| پایه‌ی تراکنش | `transaction_leg` | جزء تراکنش که موجودی (account, asset) را به‌اندازه‌ی مقدار علامت‌دار تغییر می‌دهد. |
| لات | `lot` | بسته‌ی خریدِ یک دارایی با قیمت و تاریخ مشخص؛ مبنای FIFO. |
| مصرف لات | `lot_consumption` | تخصیص یک فروش به یک یا چند لات و ثبت سود/زیان محقق‌شده. |
| قیمت | `price` | اسنپ‌شات قیمت یک دارایی برحسب یک ارز در یک تاریخ (دستی). |
| نرخ ارز | (همان `price`) | نرخ USD/IRR = قیمت دارایی فیات USD برحسب IRR. |
| بدهی | `liability` | وام/اقساط/بدهی با اصل، نرخ و برنامه‌ی بازپرداخت. |
| هدف | `goal` | هدف مالی با نوع، مقدار هدف و تاریخ سررسید. |
| اسنپ‌شات پرتفو | `portfolio_snapshot` | عکس دوره‌ای ارزش خالص برای نمودار زمانی. |
| نرخ تورم | `inflation_rate` | سری زمانی تورم. |
| فرض‌ها/تنظیمات | `assumptions` | hurdle rate، نرخ رشد فرضی هر کلاس، ارز نمایش. |

## انواع تراکنش (دقت کن قاطی نشوند)

| نوع (فارسی) | شناسه | پایه‌ها | سود/زیان؟ |
|---|---|---|---|
| واریز | `deposit` | یک پایه‌ی ورودی از «بیرون» | خیر — جریان نقدی بیرونی |
| برداشت | `withdrawal` | یک پایه‌ی خروجی به «بیرون» | خیر — جریان نقدی بیرونی |
| معامله | `trade` | دو پایه: `−A` و `+B` | بله — تولید cost basis و سود/زیان محقق‌شده |
| انتقال | `transfer` | دو پایه‌ی هم‌دارایی در دو حساب | خیر — فقط کارمزد ممکن |
| درآمد | `income` | یک پایه‌ی ورودی (سود سهام/استیکینگ/اجاره/بهره) | افزایش دارایی، نه معامله |
| کارمزد | `fee` | یک پایه‌ی خروجی | هزینه |
| هزینه | `expense` | یک پایه‌ی خروجی | هزینه |
| ارزش‌گذاری | `valuation` | بدون پایه؛ ثبت قیمت جدید برای دارایی غیرنقدشونده (ملک) | خیر |

### رویدادهای بدهی
| نوع | شناسه |
|---|---|
| دریافت وام | `liability_disbursement` |
| پرداخت قسط | `liability_repayment` |
| شناسایی بهره | `liability_interest` |

## مفاهیم محاسباتی

| مفهوم | شناسه | تعریف |
|---|---|---|
| سود/زیان محقق‌شده | `realized_pnl` | هنگام فروش: (قیمت فروش − قیمت لات) × مقدار، با FIFO. |
| سود/زیان محقق‌نشده | `unrealized_pnl` | (قیمت روز − میانگین لات‌های باز) × موجودی فعلی. |
| ارزش خالص | `net_worth` | جمع ارزش دارایی‌ها − جمع مانده‌ی بدهی‌ها. |
| بازدهی پول‌وزنی | `xirr` | بازدهی واقعی با درنظرگرفتن زمان‌بندی واریز/برداشت. |
| بازدهی زمان‌وزنی | `twr` | بازدهی استراتژی مستقل از واریز/برداشت. |
| بازدهی اسمی | `nominal_return` | بازدهی برحسب ریال بدون تعدیل تورم. |
| بازدهی حقیقی | `real_return` | بازدهی اسمی منهای تورم. |
| حداقل سود مورد انتظار | `hurdle_rate` | آستانه‌ی موفقیت؛ ثابت یا تورم یا رشد دلار. |
| انحراف تخصیص | `allocation_drift` | فاصله‌ی تخصیص فعلی از تخصیص هدف. |
